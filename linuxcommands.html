Linux Commands List
  • git add: add two files together
  • git commit: adds a file to a repository
  • git rm: removes a file from a repository
  • git mv: moves a file from one repository to another
  • git push: pushes the file as well as any changes to it to the repository
  • git pull: pulls any new changes to the file from the repository
  • ls -al: displays all the folders on the local computer
  • git status: to see the current status of the repository and displays any untracked files
  • git clone: copy a repository to your local machine
